# 元组和列表

### 列表(List)

列表是顺序存储结构『数组』在Python中的实现；列表写在方括号之间，每个元素用逗号隔开；可以被索引和切片，List还可以用 `+` 或者 `*` 操作符进行拼接，**列表中每个元素可修改**。

```python
arr = [1, 2, 'Hello']
print(arr[2]) # Hello
arr2 = arr * 2
print(arr2) # [1, 2, 'Hello', 1, 2, 'Hello']j
```

### 元组(Tuple)

与列表相似，但是元组的**元素不能修改**

```python
tup = (1, 2, 3)
print(tup[0]) # 1
tup[0] = 123 # 非法操作，报错
```
